# 🦊狐了个狗🐶

一个可爱的三消消除游戏，基于"羊了个羊"玩法的原创版本。

## ✨ 特性

- 🎮 **两关设计**
  - 第一关：极简入门（3组9张卡，只有🐶🦊🎄）
  - 第二关：进阶挑战（复杂堆叠布局，多种表情符号）
  
- 🎨 **清新设计**
  - Tiffany 蓝背景
  - 雪花飘落动画
  - 流畅的过渡动效

- 🔧 **道具系统**
  - 每关可使用一次"返还槽位"道具
  - 将未消除的卡片返回上方排列
  
- 📊 **游戏功能**
  - 实时计分系统
  - 关卡进度显示
  - 重新开始功能

## 🚀 在线体验

部署到 Vercel 后即可在线游玩！

## 🛠 技术栈

- 纯原生 HTML + CSS + JavaScript
- 无依赖，即开即用
- 响应式设计

## 📝 游戏规则

1. 点击未被遮挡的卡片移至下方槽位
2. 槽位中相同的3张卡片会自动消除
3. 消除所有卡片即可过关
4. 槽位满7张未消除则游戏失败
5. 道具每关限用一次，可救急
